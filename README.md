# Games

A retro game player inside Frontier — play classic console games right in a tab.

Games turns Frontier into a little retro arcade. Browse a library of classic console titles, pick one, and it boots up and plays in your workspace — no emulator to install, no setup. It can pull games from public online game archives so there's plenty to jump into, and you can load your own ROM files just as easily. Your progress is kept, so you can step away and pick a game back up later.

<!-- screenshot: a game running in the Games tab with the library sidebar alongside -->

## Features

- Play classic console games in a tab, straight in the browser
- A browsable library to find something to play
- Pulls titles from public online game archives
- Bring your own ROMs and play them too
- Saves are kept so you can come back to a game later
- A sidebar library and a welcome tile to get you playing fast

## Install

Install Games from the **Applications → Marketplace** tab in Frontier: find Games, click Install, and approve the access it asks for (it reaches out to the internet to fetch game catalogs). Frontier verifies the download before installing. Once it's in, the Games library appears in the sidebar — open it and start playing.
